Тестовое задание

Написать скрипт, который успешно проходит тест, расположенный по адресу: http://test.youplace.net.

Скрипт должен быть написан на языке программирования GO. Если вы еще не работали с этим языком, то вот этот туториал будет хорошей отправной точкой по языку.

Не нужно тестировать указанный сервер под нагрузкой! Этот сервер предназначен только для отладки и написания скрипта. Мы будем анализировать только код вашего скрипта.

Тестовое задание должно быть полностью выполнено к первому собеседованию. Жёстких сроков по выполнению тестового задания нет.

Скрипт должен быть помещён на любой публичный GIT репозиторий. Тестовое задание присылается на проверку виде URL ссылки на GIT репозиторий. Скрипты, присланные по почте или выложенные на файлообменники, не рассматриваются.

Желаем удачи!
